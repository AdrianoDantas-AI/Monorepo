## ADDED Requirements

### Requirement: Analise de fluxo de caixa MUST be supported in ConsoleDeGastos MVP
O sistema MUST suportar analise de fluxo de caixa conforme escopo aprovado do produto.

#### Scenario: Analise de fluxo de caixa
- **WHEN** o usuario consultar fluxo por periodo
- **THEN** o sistema MUST retornar entradas, saidas e variacao percentual
- **AND** MUST permitir comparacao temporal.
